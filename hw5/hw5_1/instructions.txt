create a generic pet and print information using PrintInfo().
create a Cat pet, use PrintInfo() to print information, and add a statement to print the cat's breed using the GetBreed() function.
Ex. If the input is:

Dobby
2
Kreacher
3
Scottish Fold
the output is:

Pet Information: 
   Name: Dobby
   Age: 2
Pet Information: 
   Name: Kreacher
